# 🌱 EcoScan – Smart City Waste & Drainage Monitoring System

> A smart web-based application to monitor, analyze, and improve urban waste and drainage systems using real-time location data.

---

## 🚀 Overview

EcoScan helps citizens and authorities report and analyze waste and drainage issues in their area.  
It uses maps, cloud storage, and charts to provide real-time insights for cleaner cities.

---

## ✨ Features

- 📍 Location-based issue reporting (via map click)
- 🗺️ Interactive map using Leaflet.js
- 📊 Waste level analysis (High / Medium / Low)
- 🔐 Secure login/signup (Firebase Auth)
- ☁️ Real-time database (Firestore)
- ⚡ Optimized performance using caching
- 📈 Visual charts (Chart.js)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Firebase Firestore  
- **Auth:** Firebase Authentication  
- **Maps:** Leaflet.js  
- **Charts:** Chart.js

## 🔥 Firebase Setup

1. Clone the repository
```bash
git clone https://github.com/kulukuricharishma/eco.git
```

2. Open the project folder

3. Add Firebase config in firebase.js

4. Run using Live Server or open index.html
## 🔥 Firebase Setup

Go to Firebase Console → Create Project → Enable:
- Authentication (Email/Password)
- Firestore Database
Then paste your Firebase configuration inside `firebase.js`
